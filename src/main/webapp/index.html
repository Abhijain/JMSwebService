Getting started with Embedded Jetty

Step 1. Setup your environment

  1. Ensure EGit is installed.
  2. Ensure the Maven Eclipse Plugin is installed.
  3. Create an SSH key:
    1. Go to Window Preferences General Network Connections SSH2
    2. Choose the Key Management tab
    3. Click Generate RSA Key...
    4. Copy the generated public key in the text box and add it to your account
    5. Click Save Private Key..., accepting the defaults
    6. Click Ok

Step 2. Clone the App

  1. Go to Window Open Perspective Other
  2. Choose Git Repository Exploring and click Ok
  3. Click Clone a Repository
  4. Choose URI and click Next
  5. Enter git@heroku.com:.git in the URI field.
  6. Click Next
  7. Click Ok to the fingerprint and known hosts prompts if they appear.
  8. Click Finish

Step 3. Import the App

  1. Choose File Import... Maven Existing Maven Projects
  2. Click Next
  3. Set the Root Directory to $HOME\git, e.g. C:\Users\steve\git
  4. Check the box next to /pom.xml
  5. Click Finish

Step 4. Makes some changes to the app

  1. Open HelloServlet.java
  2. Modify line 20 with: out.write("Deployed first change!".getBytes());

Step 5. Deploy to Heroku

  1. Right-click the project root and choose Team Share Project
  2. Choose Git
  3. Check Use or create repository in the parent folder of project
  4. Click Finish
  5. Right-click the project root and choose Team Commit
  6. Enter a commit message and click Commit
  7. Right-click the project root and choose Team Push to Upstream
  8. Review the push results. At the bottom, a "... deployed to Heroku" message will appear.

Done!

You've just cloned, modified, and deployed a brand new app.
